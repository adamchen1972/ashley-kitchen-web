# Ashley Kitchen Website Growth Plan

This document turns the current review work into an execution path for improving the English website.

Primary goals:

- make the site more beautiful
- make the site easier to understand
- make the site more commercially persuasive
- make the next step easier:
  sample request, specification request, dealer conversation, or purchase inquiry

## Core Principles

1. Beauty leads first.
2. Innovation should be visible, not only stated.
3. Product pages must feel easier to buy from, not only nicer to browse.
4. Homepage gets the strongest images.
5. Product pages get the second-best images.
6. Applications and Projects should support product selling, not compete with it.

## Phase 2: Improve Conversion

Every major page should help the visitor do one of these quickly:

- request a sample
- ask for technical specifications
- understand installation
- compare material families
- start a project conversation

### Required support content

- cleaner CTA structure
- installation guide access
- specification summary access
- compliance / certification access
- sample request path

## Phase 3: Strengthen Cross-Page Logic

### Catalog

Use it as a clean comparison page, not a stitched brochure.

Stone should become easier to browse by grouped directions such as:

- Marble Series
- Cement Series
- Sandstone Series
- Luminous Series
- Adhesive-Backed Quick Install

### Applications

Use it as a scenario-fit page.

Key future additions:

- DIY wall refresh logic for adhesive-backed 3D stone
- stronger hotel and facade proof from sandstone materials
- stronger luminous hospitality proof from the translucent series

### Projects

Use it as built proof.

Key future additions:

- international hotel proof
- stronger hospitality stone projects
- more differentiated commercial / residential / system examples

## Phase 4: Prepare For International Growth

English should remain the master version first.

Use English to stabilize:

- terminology
- page hierarchy
- asset naming
- compliance messaging
- installation support

After that, expand to other languages.
